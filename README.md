# Contact Book Application

## Tech Stack
- Backend: FastAPI (Python)
- Database: SQLite
- 
- ORM: SQLAlchemy

## Features
- Add contact
- View all contacts
- Search contact by name
- Update phone number
- Delete contact (bonus)

## Backend Setup

cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload

Open:
http://127.0.0.1:8000/docs
